Java OOP banking Application
A console based system designed to demonstrate from functional programming to Object oriented programming.

Main features
 Ecapsulation: Implemented private and public setters/getters to protect using data
 Object oriented design:Implemented using dedicated classes and objects to ensure that the code is maintable and scalable.
 secure pin logic: implements a forceful pin verification loop before the account is given access.
 Transaction management:Full support for all basic bank operation such as checking balance, deposits and withdrawals. 
 
 Technology and tools used:
 Language :java (JDK 17+)
 version control: Git & Github
 IDE: IntelliJ IDEA Community Edition
